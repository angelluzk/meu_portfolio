Reconhecimento Facial

Sistema de Controle Escolar

Sobre o Projeto

O Sistema de Reconhecimento Facial foi desenvolvido para automatizar o controle de entrada e saída de alunos em instituições de ensino, utilizando tecnologia de visão computacional.

Tecnologias Utilizadas

  • Python e OpenCV
  • MySQL para armazenamento de dados
  • Interface gráfica com Tkinter

Funcionalidades

  • Cadastro e reconhecimento facial de alunos.
  • Registro automático de presença.
  • Relatórios de frequência.
  • Segurança e autenticação de usuários.
