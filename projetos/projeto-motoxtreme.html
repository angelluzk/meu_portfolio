MotoXtreme

Sistema de Vendas

Sobre o Projeto

O MotoXtreme é um sistema para gerenciamento de vendas e estoque em lojas de motocicletas. Desenvolvido para melhorar processos internos e otimizar a experiência do cliente.

Tecnologias Utilizadas

  • HTML, CSS e JavaScript
  • PHP
  • MySQL
  • Bootstrap

Funcionalidades

  • Cadastro de produtos e categorias.
  • Gerenciamento de vendas e estoque.
  • Relatórios financeiros.
  • Painel administrativo.
