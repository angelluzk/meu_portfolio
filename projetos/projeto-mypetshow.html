MyPetShow

Sistema para PetShop

Sobre o Projeto

O MyPetShow é um sistema de gerenciamento para PetShops, facilitando o cadastro de clientes, pets e serviços. Foco na organização e eficiência no atendimento.

Tecnologias Utilizadas

  • HTML, CSS e JavaScript
  • PHP com Laravel
  • MySQL
  • Bootstrap

Funcionalidades

  • Cadastro de clientes e pets.
  • Gerenciamento de agendamentos.
  • Controle de estoque de produtos.
  • Emissão de relatórios.
