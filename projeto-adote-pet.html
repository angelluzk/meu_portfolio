Adote Pet

Sistema de Adoção de Animais

Sobre o Projeto

O Adote Pet é uma plataforma web que visa promover a adoção responsável de animais. Utilizando design empático e tecnologia, busca criar uma ponte entre abrigos e pessoas interessadas em adotar.

Tecnologias Utilizadas

  • HTML, CSS e JavaScript
  • PHP para backend
  • MySQL para banco de dados
  • Bootstrap para estilização

Funcionalidades

  • Cadastro de animais disponíveis para adoção.
  • Sistema de busca e filtro por espécie, raça e porte.
  • Cadastro de usuários interessados.
  • Painel administrativo para gerenciamento.
